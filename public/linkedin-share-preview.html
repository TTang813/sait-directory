1 How it works

  • Member completes Opt-in Step 2, then clicks "Share on LinkedIn"
  • The page opens LinkedIn's native share dialog pre-filled with their name, specialisations, and the directory URL
  • LinkedIn automatically generates the preview card from the URL's Open Graph (OG) tags
  • No image file is needed — LinkedIn builds the card from text data
The card thumbnail comes from a pure CSS background on the shared URL — no image file to manage or upload.

2 OG metadata injected on the practitioner page

  • og:title — "Jane Smith — SAIT Practitioner Directory"
  • og:description — Region, top 3 specialisations, PR number
  • og:image — A server-rendered HTML page styled to look like a branded card, served as a PNG via Next.js og:image endpoint
  • og:url — The practitioner's unique directory URL
